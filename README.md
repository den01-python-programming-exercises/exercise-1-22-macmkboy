# Exercise 1.22 - Ancient

Write a program that prompts the user for a year. If the user inputs a number that is smaller than 2015, then the program prints the string "Ancient history!".

```plaintext
Give a year:
*2017*
```

```plaintext
Give a year:
*2013*
Ancient history!
```
